TankViewer (version 1.02)

TankViewer is a little utility to view and extract contents from the resource files (tank) used by Gas Powered Games' Dungeon Siege.

History

v 1.0 (April 7, 2002)
  • Support for multiple tanks open at once.
  • Sort files either ascending or descending by Name, File type, size or Last modified date.
  • Built in text, hex and image viewers. So you can just doubble click on a file in tankviewer to view it without saving it to the disk first.
  • Associate .dsmap .dsres .dssave .dsasave .dsqsave .dsparty so that you can just doubbleclick on them in explorer to open them in TankViewer
  • Drag and Drop support: Just drag a tank file from explorer and drop it on TankViewer to open it. And to extract files just drag them from TankViewer to them destination folder in explorer.
  • Can open and export .tvc files, a list of several tankfiles so that you easily can open all those tanks just by doubble clicking on the .tvc file.
v 1.02 (May 11, 2002)
  • Fixed filelist key shortcut "Index out of bounds" error. I.E when you pressed "E" it would step through all files starting with the name "E", just like Explorer.
  • File type now fefaults to " File" instead of "Unknown".
  • The options now have a button to set the startup directory to where Dungeon Siege is installed. Note that the button will only be visible id DS is installed.
  • Added a placeholder plugin with icons and descriptions for various formats.
Issues

There are a few known issues with this version. Like the performance when extracting alot of files, for example all the 300MB in Objects.dsres. This will be fixed in a later build.


Some shortcuts.

Ctrl+OShows the open tank dialog.
Ctrl+F4Closes active tank
Shift+EOpens the extract dialog
F1Tank Info
BackspaceGo up one directory
EnterOpen file or directory
Shift+VSame as enter (but also works when the listview doesn't have focus)
Ctrl+ASelect all files and folders
Ctrl+Shift+IInvert selection
